CATIA

REGISTER

  • Description: The competition is to test how fast one interprets the given problem using CATIA.
    Entry fee: Rs.50/-
    Number of participant(s): 01
  • RULES:
    • The competition is to test how fast one interprets the given problem using CATIA.
    • Catia enables you to quickly create design models with an intuitive, model centric interface. You can create, revise models in graphics window using tool bars and dashboards.
    • Equally important Catia interface components include the navigator and browser, which enable you to quickly locate, preview, and open design model.
    • You also have the ability to manipulate and orient design models, which is an important basic skill that you will use whenever you are working in Catia .
    • First round: In this round, candidates are given basic models to test their command mastery and speed in modelling. The one completing the task in minimum time will proceed to next round.
    • Second round: In this round, candidates will have to create several parts, and use the minimum assembly design.
    • The winner will be decided on basis of accuracy of sketch and time required to assemble given task.
    • Participants are requested to be present at the venue 20 mins before the reporting time.
    • It is mandatory for every participant to carry their respective receipts without which entry will not be permitted.
    • Every participant should have his/her own valid I-Card.
    • Decision of the judges and organizing team will be final and binding.
  • OBJECTIVES:
    • Identify core Catia concepts
    • Navigate the main Catia tools and features
    • Preview, open, and orient Design model
    • Manage Catia memory and directories
